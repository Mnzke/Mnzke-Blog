树状数组

发表于 2021-12-09 更新于 2021-12-11 分类于 ACM 知识点 ， 数据结构

树状数组

树状数组能解决的问题：

给定一个长度为n的数组，多次进行如下两种操作：

  1. 给定一个数m，得到数组1-m的前缀和。
  2. 对某个指定下标的数值进行修改

根据我们的一般思想，可能就是建立前缀和数组，每次去更新数的时候，再去更新一次前缀和数组，建立前缀和数组\Omicron(n)而更新一次最坏也是\Omicron(n)，这个时间复杂度是无法接受的，所以便出现了树状数组。

树状数组

如图，对于一个长度为n的数组，A数组存放的是数组的初始值，引入一个辅助数组C，即为树状数组；

C[1]=A[1]C[1] = A[1]C[1]=A[1]
C[2]=A[1]+A[2]C[2] = A[1] + A[2]C[2]=A[1]+A[2]
C[3]=A[3]C[3] = A[3]C[3]=A[3]
C[4]=A[1]+A[2]+A[3]+A[4]C[4] = A[1] + A[2] + A[3] + A[4]C[4]=A[1]+A[2]+A[3]+A[4]
C[5]=A[5]C[5] = A[5]C[5]=A[5]
C[6]=A[5]+A[6]C[6] = A[5] + A[6]C[6]=A[5]+A[6]
C[7]=A[7]C[7] = A[7]C[7]=A[7]
C[8]=A[1]+A[2]+A[3]+A[4]+A[5]+A[6]+A[7]+A[8]C[8] = A[1] + A[2] + A[3]+ A[4] + A[5] + A[6] + A[7] + A[8]C[8]=A[1]+A[2]+A[3]+A[4]+A[5]+A[6]+A[7]+A[8]

我们从中可以发现这样的规律，我们称C[i]C[i]C[i]的值为下标为iii的数所管辖的数的和，下标为iii的数所管辖的元素的个数为2k2^k2​k​​ 个（kkk为iii的二进制的末尾000的个数）：

  • i=8=1000i = 8 = 1000i=8=1000，末尾333个000，故k=3k = 3k=3，所管辖的个数为23=82^3 = 82​3​​=8 ，C[8]C[8]C[8]是888个数的和；
  • i=5=0101i = 5 = 0101i=5=0101，末尾没有000，故k=0k = 0k=0，所管辖的个数为 20=12^0 = 12​0​​=1，C[5]C[5]C[5]是111个数的和；

而对于前m个数的和也同样可以看出规律

C[1]=A[1]C[1] = A[1]C[1]=A[1]
C[2]=C[1]+A[2]C[2] = C[1] + A[2]C[2]=C[1]+A[2]
C[3]=A[3]C[3] = A[3]C[3]=A[3]
C[4]=C[2]+C[3]+A[4]C[4] = C[2] + C[3] + A[4]C[4]=C[2]+C[3]+A[4]
C[5]=A[5]C[5] = A[5]C[5]=A[5]
C[6]=C[5]+A[6]C[6] = C[5] + A[6]C[6]=C[5]+A[6]
C[7]=A[7]C[7] = A[7]C[7]=A[7]
C[8]=C[4]+C[6]+C[7]+A[8]C[8] = C[4]+ C[6] + C[7] + A[8]C[8]=C[4]+C[6]+C[7]+A[8]

我们可以看出 SUMi=Ci1+Ci2+...SUM_i=C_{i1}+C_{i2}+...SUM​i​​=C​i1​​+C​i2​​+...,而这里m和C[i]的对应关系是这样的，对于查询的m，将它转换成二进制后，不断对末尾的1的位置进行-1的操作，直到全部为0停止，中间得到的值就是c[i]，例如：

m=7m = 7m=7，SUM7=C7+C6+C4SUM_7 = C_7 + C_6 + C_4SUM​7​​=C​7​​+C​6​​+C​4​​，777的二进制为( C7C_7C​7​​ 得到)，对011101110111的末尾1的位置-1，得到0110 = 6（C6C_6C​6​​得到），再对011001100110末尾111位置-1，得到0100=40100 = 40100=4（$ C_4$ 得到），最后对010001000100末尾111位置−1-1−1后得到000000000000（结束），计算停止，至此$ C_7，C_6，C_4$全部得到，求和后就是m = 7时它的前缀和；

那么现在我们进行更新的操作

假设更新A[2]=5A[2] = 5A[2]=5，通过观察我们得知，如果修改了A[2]A[2]A[2]的值，那么管辖A[2]A[2]A[2]的C[2]，C[4]，C[8]的前缀和都要加上5（所有的祖先节点），那么和查询类似，我们如何得到C2C2C2的所有祖先节点呢，依旧是上述的巧妙的方法，但是我们把它倒过来用，对于要更新i位置的值，我们把iii转换成二进制，不断对二进制最后一个1的位置+1，直到达到数组下标的最大值n结束，对于给出的例子i = 2，假设数组下标上限n=8n = 8n=8，iii转换成二进制后等于0010（$ C_2），对末尾1的位置进行+1，得到0100（ C_4$ ），对末尾的1的位置进行+1，得到1000（$ C_8$ ），循环结束，对$ C_2，C_4，C_8的前缀和都要+5，当然不能忘记对A[2]$的值+5，单点更新值过程结束。

所以我们在建树的时候就可以理解成对一个全是0的数据不断进行更新操作。

代码

\begin{aligned}
f_Y(y) & = f_X[h(y)]|h’(y)| \[2ex]
& = f_X[h(y)]h’(y) \[2ex]
& = \frac{1}{\theta}e^{-\frac{x}{\theta}}[\frac{dx}{dy}(-\frac{\theta}{ln(1-y)})] \[2ex]
& = \frac{1}{\theta}e^{-\frac{-\frac{\theta}{ln(1-y)}}{\theta}}\frac{\theta}{1-y} \[2ex]
& = \frac{1}{\theta}e^{ln(1-y)}\frac{\theta}{1-y} \[2ex]
& = \frac{1-y}{\theta}\frac{\theta}{1-y} \[2ex]
& = 1
\end{aligned}
